# ALX Travel App 0x00

This project implements a Django-based system for managing travel listings, handling user bookings, and collecting reviews.

## Models Overview

- **Listing**: Represents a travel destination, including pricing details.
- **Booking**: Stores reservation records made by users.
- **Review**: Contains user-submitted feedback for each listing.

## Database Seeding

Populate the database with sample data by running:

```bash
python manage.py seed
```
